Linux下Crontab语法详解

发表于 2016-08-31   |   分类于 Linux   |     |   views

Linux下计划任务Crontab用法详解。

基本格式

  • 第一列表示分钟: 使用数字 0-59 表示
  • 第二列表示小时: 使用数字 0-23 表示
  • 第三列表示日期: 使用数字 1-31 表示
  • 第四列表示月份: 使用数字 1-12 表示
  • 第五列表示星期: 使用数字 0-7 表示, 0 和 7 均代表星期日
  • 第六列表示要执行的命令
